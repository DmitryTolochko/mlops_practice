# Модель для сравнения текстов на схожесть.

## Испольован датасет: https://www.kaggle.com/datasets/rishisankineni/text-similarity/code

## Запуск

* Открыть папку на linux/macOS
* Запустить терминал в папке
* Написать ```chmod +x pipeline.sh```
* Написать ```./pipeline.sh```

Ожидаемый результат: ```Model test accuracy is: 0.911```
